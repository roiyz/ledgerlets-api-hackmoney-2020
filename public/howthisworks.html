What is ledgerlets?

Ledgerlets is a service that lets you create proof that you had a specific document in a specific point in time.
The proof is generated without disclosing the contents of the actual document and is stored on the Ethereum blockchain so that it can't be disputed or be taken down by private or governmental agencies.
It's also FREE!

How can this service be used?
  • Contracts time stamping and formalization
    • Supply chain tracking
  • Investigative reporters wanting to establish evidence without disclosing the data
  • Turn based games - “Diplomacy” as one example
  • And more

Instructions:
  • 1. Upload the file you want to use as proof on the main page.
    Don't worry, the file contents is not being uploaded, only a unique key generated by the file's content (known as a digest) as well as the file's size are sent over.
  • 2. once uploaded, digests from multiple documents are collected into one file and that file is being used to save the information in the Ethereum blockchain.
  • 3. Save the hashes file provided once the process is complete. This is mandatory to prove existence in the future
  • To prove existence of your document, just upload it again to ledgerlets.com. It will show you exactly when the file was initially submitted. if ledgerlets.com is not available. Perform the following:
    a. get the sha1 of the hashes file (the file mentioned in section 2 above)
    b. use the value of the file above to look for the same address on the blockchain.
    c. the time of the earliest transaction states when your document was initially submitted
    d. bonus - If there's a concern for SHA1 collision, the transaction mentioned in item c should also have the SHA256 value as part of the data field.
